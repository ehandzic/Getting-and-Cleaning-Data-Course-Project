# Getting and Cleaning Data Course Project

Data for this project was downloaded from this site http://archive.ics.uci.edu/ml/datasets/Human+Activity+Recognition+Using+Smartphones  
This is link to data set file : https://d396qusza40orc.cloudfront.net/getdata%2Fprojectfiles%2FUCI%20HAR%20Dataset.zip

The data represent data collected from the accelerometers from the Samsung Galaxy S smartphone.

## The run_analysis.R script performs the following steps to clean the data:
* Downloads required data from
* Unzip data
* Merges the training and the test sets to create one data set
* Extracts only the measurements on the mean and standard deviation for each measurement
* Uses descriptive activity names to name the activities in the data set
* Write tidy data set with the average of each variable for each activity and each subject to output file
